Windows Terminal安装、美化与在当前路径打开

Posted on 2020-12-26 In Tools
Symbols count in article: 4.5k Reading time ≈ 4 mins.

安装

在Win 10应用商店搜索Windows Terminal即可安装。

美化

具体配置教程可前往官方文档查看。

右键在当前位置打开

手动配置

  1. Win+R 键，输入regedit，运行注册表编辑器；

  2. 手动查找或在注册表窗口上面直接输入\HKEY_CLASSES_ROOT\Directory\Background\shell找到这个注册表路径；

  3. 选中shell文件，右键新建项，命名为wt，并设置(默认)值为你想要的名称，如 Windows Terminal Here；

  4. 再新建字符串值Icon即显示图标，并将值设设置C:\Program Files\WindowsApps\Microsoft.WindowsTerminal_<version>\WindowsTerminal.exe

    注意：值看你安装的具体版本，可通过Everything搜索查证，或者设置为cmd的图标C:\Windows\System32\cmd.exe免去更新的麻烦，再或者设置自己的图标，不影响使用。

  5. 如需要将这个命令置顶，则添加新字符串值Position，值为Top

  6. 右键新建项command，默认值cmd /c set CURRENT_PATH="%V" & start C:\Users\[username]\AppData\Local\Microsoft\WindowsApps\wt.exe

    这里的路径是Terminal的路径，一般将[username]修改成自己的用户名即可。

  7. 打开设置后在profiles>list>中选取你默认的终端，一般默认是Windows PowerShell，在其中添加一行配置"startingDirectory":null，目的是从当前目录打开。
